### 五、项目介绍

项目编号：springbootA206D

口红销售系统是管理和优化口红产品从销售、库存到客户服务的全流程，以提升效率和销售额的工具。

角色：管理员、用户、商家

管理员：系统首页、商家、用户、口红类型、颜色、口红信息、充值记录、系统管理、订单管理、个人中心。

用户：首页、口红信息、公告信息、购物车、个人中心 修改密码 聊天记录 我的订单 我的地址 我的收藏 我的优惠券。

商家：系统首页、红包信息、优惠券、订单管理、个人中心。
